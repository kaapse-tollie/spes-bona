# Derived Outputs

These files are generated output. Rebuild them with the public CLI.

## Generated Output

- `final_resource_caps.csv`
  - current audited caps by state/resource
- `resource_adjustments.csv`
  - full `X + Y - Z` reconstruction surface
- `resource_denominators.csv`
  - family denominators
- `regional_resource_totals.csv`
  - regional rollups
- `state_delta_summary.csv`
  - one row per SB state
- `state_resource_deltas.csv`
  - one row per `state × resource`
- `growth_series.csv`
  - generated growth tables
- `target_observations.csv`
  - public target observation surface for non-arable families
- `comparator_observations.csv`
  - public comparator observation surface for non-arable families
- `gdp_selection_rows.csv`
  - GDP year-selection audit for non-arable families
